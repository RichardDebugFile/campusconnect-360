CampusConnect 360
Ecosistema activo Red de colegios

Un día de operación, de punta a punta

Cada portal representa un actor del colegio. Sus acciones publican eventos vía RabbitMQ que sincronizan todo el ecosistema en tiempo real.

Flujo de integración recomendado

1. Académico 2. Financiero 3. Docente / Bienestar 4. Dashboard Directivo
Todos los usuarios de demostración usan la clave: campus123

Portal Académico

Registro y gestión administrativa de alumnos, con control centralizado de matrículas.

Entrar como Secretaría pub: StudentEnrolled

Portal Financiero

Generación de obligaciones y conciliación de pagos con confirmación de transacciones.

Entrar como Finanzas pub: PaymentConfirmed

Portal Docente / Bienestar

Registro diario de asistencias y reporte ágil de incidentes académicos o conductuales.

Entrar como Docente pub: Attendance / Incident

Dashboard Directivo

Consolidado de indicadores, control analítico global (CQRS) y demostración de resiliencia.

Entrar como Admin / Director consume: todo (CQRS)

Recursos técnicos e integración

API Gateway http://localhost:8080
RabbitMQ (consola) guest / guest http://localhost:15672
Swagger por servicio /api/<servicio>/docs
